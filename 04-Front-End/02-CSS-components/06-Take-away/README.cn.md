## 你的CSS前途一片光明!

恭喜你，今天是你CSS生涯中非常重要的一天。

从现在开始：

- 你可以使用[Le Wagon UI 工具](https://uikit.lewagon.com/)，就不用每次都重新编码简单的组件😊。
- 记住，一定要**花时间考虑你的HTML结构**（wrapper包装器、container容器、内部组件等）。这是**90%的工作**。一旦你完成了这些工作，你只需要重复使用组件和做一些小的CSS调整。
